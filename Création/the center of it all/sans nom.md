## Structure de référence
| Section           | Durée (mes.) | Remarques |
| ----------------- | ------------ | --------- |
| Intro (idée 1)    | 8            |           |
| "drop"            | 8            |           |
| Pont (idée 2)     | 8            |           |
| Variation du pont | 8            |           |
| verse (idée 3)    | 4            |           |
| variation         | 12           |           |
| Outro (idée 3)    | 4            |           |

- Cette structure laisse libre cours aux idées des compositeurs. On peut la qualifier de rhapsodique. Cela signifie que les idées s'enchaînent et sont en évolution perpétuelle. C'est une structure propre à une certaine conception de la musique électronique, généralement associée à l'idm.
